# Contributing to Shuttle

Thank you for your interest in contributing to Shuttle!

## Ways to Contribute

### Report Issues
Found a bug or have a feature request? [Open an issue](../../issues/new/choose) using our templates.

### Share Knowledge
Have a loom or content you'd like to share with the community? Submit a pull request to the `knowledge/` folder.

### Improve Documentation
Help us improve tutorials and documentation by submitting pull requests.

## Guidelines

- Be respectful and constructive
- Search existing issues before creating new ones
- Provide as much detail as possible when reporting bugs
- For feature requests, explain the problem you're trying to solve

## Questions?

Feel free to open a discussion or issue if you have questions.
